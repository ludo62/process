3. Configuration de VS Code

Pour ajouter ces configurations à votre fichier settings.json dans VS Code, suivez ces étapes :

  1. Ouvrez VS Code
  2. Appuyez sur Cmd + Shift + P (sur Mac) ou Ctrl + Shift + P (sur Windows/Linux) pour ouvrir la palette de commandes
  3. Tapez "settings" dans la palette de commandes
  4. Sélectionnez "Preferences: Open User Settings (JSON)"
  5. Dans le fichier JSON qui s'ouvre, ajoutez ou modifiez les configurations suivantes : 
        {
          "editor.defaultFormatter": "esbenp.prettier-vscode",
          "editor.formatOnSave": true,
          "editor.codeActionsOnSave": {
            "source.fixAll.eslint": "explicit",
            "source.organizeImports": "explicit"
          },
          "emmet.triggerExpansionOnTab": true,
          "typescript.updateImportsOnFileMove.enabled": "always",
          "typescript.inlayHints.variableTypes.enabled": true,
          "typescript.inlayHints.parameterTypes.enabled": true,
          "typescript.inlayHints.parameterNames.enabled": "literals",
          "[liquid]": {
            "editor.defaultFormatter": "Shopify.theme-check-vscode",
            "editor.formatOnSave": true
          }
        }

Ces configurations optimiseront votre environnement de développement pour les extensions installées.

4. Snippets VS Code utiles

Voici quelques snippets utiles que vous pouvez ajouter à VS Code pour accélérer votre développement, en particulier pour les sections de schéma Shopify Liquid :

  1. Ouvrez VS Code
  2. Appuyez sur Cmd + Shift + P (sur Mac) ou Ctrl + Shift + P (sur Windows/Linux)
  3. Tapez "snippets" et sélectionnez "Preferences: Configure User Snippets"
  4. Choisissez "New Global Snippets file..." ou sélectionnez un fichier de snippets existant pour le langage souhaité (par exemple, liquid.json pour Liquid)
  5. Ajoutez les snippets suivants :

Snippet pour une nouvelle section schema : 

              {
                "New Section Schema": {
                  "prefix": "newsection",
                  "body": [
                    "{% schema %}",
                    "{",
                    "  \"name\": \"${1:Section name}\",",
                    "  \"settings\": [",
                    "    $0",
                    "  ],",
                    "  \"presets\": [",
                    "    {",
                    "      \"name\": \"${2:$1}\",",
                    "      \"category\": \"${3:Custom}\"",
                    "    }",
                    "  ]",
                    "}",
                    "{% endschema %}"
                  ],
                  "description": "Create a new section schema with presets"
                }
              }

Snippet pour un paramètre de texte : 

              {
                "Text Setting": {
                  "prefix": "textsetting",
                  "body": [
                    "{",
                    "  \"type\": \"text\",",
                    "  \"id\": \"${1:text_id}\",",
                    "  \"label\": \"${2:Text Label}\",",
                    "  \"default\": \"${3:Default Text}\"",
                    "}${0}"
                  ],
                  "description": "Add a text setting to schema"
                }
              }

Snippet pour un paramètre d'image : 

              {
                "Image Setting": {
                  "prefix": "imagesetting",
                  "body": [
                    "{",
                    "  \"type\": \"image_picker\",",
                    "  \"id\": \"${1:image_id}\",",
                    "  \"label\": \"${2:Image Label}\"",
                    "}${0}"
                  ],
                  "description": "Add an image picker setting to schema"
                }
              }

Snippet pour un paramètre de type range : 

              {
                "Range Setting": {
                  "prefix": "rangesetting",
                  "body": [
                    "{",
                    "  \"type\": \"range\",",
                    "  \"id\": \"${1:range_id}\",",
                    "  \"label\": \"${2:Range Label}\",",
                    "  \"min\": ${3:0},",
                    "  \"max\": ${4:100},",
                    "  \"step\": ${5:1},",
                    "  \"default\": ${6:50},",
                    "  \"unit\": \"${7:px}\"",
                    "}${0}"
                  ],
                  "description": "Add a range setting to schema"
                }
              }

Snippet pour un paramètre de type select : 

              {
                "Select Setting": {
                  "prefix": "selectsetting",
                  "body": [
                    "{",
                    "  \"type\": \"select\",",
                    "  \"id\": \"${1:select_id}\",",
                    "  \"label\": \"${2:Select Label}\",",
                    "  \"options\": [",
                    "    {",
                    "      \"value\": \"${3:option1}\",",
                    "      \"label\": \"${4:Option 1}\"",
                    "    },",
                    "    {",
                    "      \"value\": \"${5:option2}\",",
                    "      \"label\": \"${6:Option 2}\"",
                    "    }",
                    "  ],",
                    "  \"default\": \"${7:$3}\"",
                    "}${0}"
                  ],
                  "description": "Add a select setting to schema"
                }
              }

Snippet pour un paramètre de type color : 

              {
                "Color Setting": {
                  "prefix": "colorsetting",
                  "body": [
                    "{",
                    "  \"type\": \"color\",",
                    "  \"id\": \"${1:color_id}\",",
                    "  \"label\": \"${2:Color Label}\",",
                    "  \"default\": \"${3:#000000}\"",
                    "}${0}"
                  ],
                  "description": "Add a color setting to schema"
                }
              }

Pour utiliser ces snippets, tapez simplement le préfixe (par exemple, "newsection") dans un fichier Liquid et appuyez sur Tab. Le snippet s'expandera automatiquement, vous permettant de remplir rapidement les champs nécessaires.
